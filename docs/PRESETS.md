# Presets

Presets are applied through CLI: `--preset <name>`.

## strict-privacy

Goal: maximum privacy.

Effects:
- DNS fallback: DoH only (no system DNS).
- ECH mode: `real`.
- Evasion: `fragment`.
- Privacy tracker blocker: enabled.
- Referer policy: enabled + `strip`.
- Signals: `DNT=1`, `Sec-GPC=1`.

## balanced-privacy

Goal: practical privacy with lower breakage risk.

Effects:
- Tracker blocker: disabled.
- Referer policy: enabled + `origin_only`.
- Signals: `Sec-GPC=1`, `DNT=0`.

## max-compatibility

Goal: maximum compatibility.

Effects:
- System DNS fallback allowed.
- ECH mode: `grease`.
- Evasion disabled.

## aggressive-evasion

Goal: anti-censorship first.

Effects:
- DNS chain: DoH + DoT + DoQ.
- ECH mode: `auto`.
- Evasion: `auto` + split offsets + traffic shaping.

Notes:
- Domain fronting rules are not auto-generated by presets.
